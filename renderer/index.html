Architecture Features

  • Single Window Design: One main window for all functionality
  • Worker Thread Pool: CPU-intensive tasks run in parallel worker threads
  • Child Processes: Support for spawning Node.js child processes for isolation
  • Plugin System: Modular architecture for easy extension
  • High Performance: Optimized for high-end PCs with multiple cores
  • IPC Communication: Secure communication between processes via callbacks

Developer Information

This application demonstrates a scalable Electron architecture with:

  • Main process for application lifecycle and system operations
  • Single renderer process for UI (this window)
  • Worker thread pool for parallel computation
  • Child process support for heavy/isolated tasks
  • Plugin system for modular functionality

Note: Add custom plugins to the plugins/ directory to extend functionality.
